    Animation Module¶

    pptx_components.components.animation ¶

    Animation and transition effects for pptx_components.

    Note: python-pptx does not natively support slide animations/transitions. This module provides helper functions to simulate animation effects by: 1. Creating duplicate shapes with staged opacity/positioning 2. Providing preset animations (fade, slide, grow) 3. Enabling manual multi-frame rendering for slide-show animation

    For true PowerPoint animations, use the PowerPoint GUI or python-pptx-animators package. This module is useful for procedural effect generation, testing, and visual effects.

    AnimationConfig dataclass ¶

    Configuration for an animation effect.

    Attributes:

    Name Type Description
    duration_ms int

    Total animation duration in milliseconds.
    frames int

    Number of keyframes to generate (for multi-frame export).
    easing str

    Easing function name ("linear", "ease_in", "ease_out", "ease_in_out").

    FadeInEffect ¶

    Bases: Component

    Fade-in animation overlay (opacity ramp from 0 to 1).

    Renders the underlying component with opacity starting at 0 and ending at 1. In static PowerPoint, this renders at full opacity. For animation, use the config.frames property to generate keyframe layers for slide-show export.

    Parameters:

    Name Type Description Default
    child Component

    Component to fade in.

    		 required
    config AnimationConfig | None

    AnimationConfig with duration and easing settings.

    		 None
    Example

    fade = FadeInEffect( ... SectionHeader("Welcome", badge_text="Intro"), ... AnimationConfig(duration_ms=1000, frames=20, easing="ease_out") ... ) builder.add(fade, w=8, h=0.5)

    GrowEffect ¶

    Bases: Component

    Scale-up animation (size expansion from 0 to full).

    Animates component from small/center-point to full size.

    Parameters:

    Name Type Description Default
    child Component

    Component to grow.

    		 required
    config AnimationConfig | None

    AnimationConfig with duration and easing settings.

    		 None
    Example

    grow = GrowEffect( ... BarChart( ... ["Jan", "Feb", "Mar"], ... [[10, 12, 15]], ... title="Revenue Trend" ... ), ... config=AnimationConfig(duration_ms=1200, easing="ease_out") ... )

    SlideInEffect ¶

    Bases: Component

    Slide-in animation from edge (position translation).

    Animates component from off-screen into final position.

    Parameters:

    Name Type Description Default
    child Component

    Component to slide in.

    		 required
    direction str

    "left", "right", "top", or "bottom".

    		 'left'
    config AnimationConfig | None

    AnimationConfig with duration and easing settings.

    		 None
    Example

    slide_in = SlideInEffect( ... MetricCard("Revenue", "$5.2M", "+12%", True), ... direction="left", ... config=AnimationConfig(duration_ms=800, easing="ease_out") ... )
