    ScatterPlot¶

    pptx_components.components.scatter.ScatterPlot ¶

    Bases: Component

    XY scatter plot visualization for correlation and distribution analysis.

    Renders data points as circles on a 2D grid. Useful for: - Correlation analysis (feature vs. performance) - Outlier detection - Risk/return quadrants - Portfolio positioning

    Axes
    • X-axis: Horizontal numeric range
    • Y-axis: Vertical numeric range
    • Optional quadrant dividers and labels
    • Configurable axis labels and tick marks
    Data Points
    • Each point: (x, y, size=0.15, color=theme.ACCENT)
    • Optional point labels (text annotation)
    • Optional grouping by color

    Parameters:

    Name Type Description Default
    points list[tuple[float, float, str | None, tuple[int, int, int] | None, float]]

    List of (x, y, label=None, color_rgb=None, size=0.15) tuples. x, y: Numeric coordinates (will be scaled to grid). label: Optional text label for point. color_rgb: Optional (R, G, B) point color; theme.ACCENT used if None. size: Point radius in inches. Default: 0.15 (small dot).

    		 required
    x_label str | None

    X-axis label string (e.g. "Features", "Cost").

    		 None
    y_label str | None

    Y-axis label string (e.g. "Performance", "Value").

    		 None
    title str | None

    Optional plot title.

    		 None
    x_range tuple[float, float] | None

    (min, max) numeric range for x-axis. Auto-calculated if None.

    		 None
    y_range tuple[float, float] | None

    (min, max) numeric range for y-axis. Auto-calculated if None.

    		 None
    show_grid bool

    Whether to draw gridlines. Default: True.

    		 True
    quadrant_labels tuple[str, str, str, str] | None

    (top_left, top_right, bottom_left, bottom_right) text labels for quadrants.

    		 None
    Example

    scatter = ScatterPlot( ... points=[ ... (2.5, 8.0, "Algorithm A", (100, 200, 255), 0.2), ... (5.0, 6.5, "Algorithm B", (100, 200, 255), 0.2), ... (3.0, 4.0, "Outlier", (255, 100, 100), 0.25), ... ], ... x_label="Complexity", ... y_label="Accuracy (%)", ... title="Model Performance Landscape", ... quadrant_labels=("High Perf", "Ideal", "Low Complexity", "Avoid") ... )

    WaterfallChart¶

    pptx_components.components.waterfall.WaterfallChart ¶

    Bases: Component

    Cumulative delta (waterfall) chart built from themed rectangles.

    Each bar floats at the cumulative level, showing a positive (POSITIVE color) or negative (NEGATIVE color) contribution. An optional Total bar spans from zero to the final sum and is rendered in ACCENT color.

    No external chart library required — uses pptx rectangles directly.

    Parameters:

    Name Type Description Default
    categories list[str]

    Label for each delta step.

    		 required
    values list[float]

    Positive or negative change at each step.

    		 required
    title str | None

    Optional heading above the chart.

    		 None
    show_total bool

    Append a summary "Total" bar.

    		 True
    total_label str

    Label for the total bar.

    		 'Total'
